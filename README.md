# CLI .NET

- **Dotnet CLI Cheat Sheet** (https://cheatography.com/oba/cheat-sheets/dotnet-cli/)


**Run application**
```
dotnet run
```


**Restore dependencies**
```
dotnet restore
```



**Build the application**
```
dotnet build
```



**Add a NuGet package**
```
dotnet add package <PackageName>
```



**List installed SDKs**
```
dotnet --list-sdks
```
